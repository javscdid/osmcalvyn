📖 How to Play

📱 Android

  1. Download dan install PowerTunnel atau Virtual Host.
  2. Import file host server.
  3. Aktifkan aplikasi, lalu buka Growtopia.

🍎 iOS

  1. Install Surge 5 dari App Store.
  2. Tambahkan konfigurasi host server.
  3. Aktifkan Surge, lalu buka Growtopia.

💻 PC / Laptop

  1. Buka file C:\Windows\System32\drivers\etc\hosts.
  2. Tambahkan baris IP server.
  3. Simpan file, lalu jalankan Growtopia.
