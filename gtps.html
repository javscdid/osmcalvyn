Tutorial

Android (PowerTunnel)

https://gtpshost.com/raw/OverallPS-GH
  • Download aplikasi PowerTunnel (APK) dari sumber tepercaya.
  • Buka PowerTunnel → menu (≡) → Host Settings.
  • Pilih Import atau Paste host, lalu tempel URL yang disalin.
  • Set update period ke on start atau sesuai instruksi host.
  • Aktifkan PowerTunnel lalu buka Growtopia, server akan muncul.

iOS (Surge 5)

https://gtpshost.com/ios/OverallPS-GH
  • Install Surge 5 dari App Store.
  • Buka Surge → Import / Edit konfigurasi → paste host URL yang disalin.
  • Simpan konfigurasi, aktifkan, lalu jalankan Growtopia.
  • Jika butuh, beri nama konfigurasi agar mudah dipilih kembali.

PC / Laptop

yourserverip.com:17091
  • Buka folder sistem Growtopia atau folder network tool yang kamu gunakan.
  • Edit file hosts (Windows: C:\Windows\System32\drivers\etc\hosts).
  • Tambahkan baris IP server (contoh: 123.45.67.89 yourserverip.com), simpan.
  • Jalankan Growtopia lalu login ke server.
